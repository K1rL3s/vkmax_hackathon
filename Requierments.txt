# Требования к проекту

Этот проект представляет собой полнофункциональное приложение, разработанное с использованием Python для бэкенда и TypeScript/React для фронтенда. Все приложение предназначено для запуска с использованием Docker.

## Бэкенд (Python)

Бэкенд построен с использованием фреймворка FastAPI и состоит из нескольких компонентов: веб-API, Telegram-бота и планировщика задач.

**Фреймворки и библиотеки:**
- **Веб-фреймворк:** FastAPI
- **Асинхронная очередь задач:** Taskiq
- **ORM:** SQLAlchemy 2.0 с Alembic для миграций
- **Драйвер базы данных:** Psycopg (для PostgreSQL)
- **Хранилище в памяти:** Redis
- **Сервер приложений:** Uvicorn / Granian
- **Пользовательские библиотеки:**
  - `maxo`: Пользовательская библиотека для разработки ботов.
  - `retejo`: Пользовательская библиотека для работы с сетью.
- **Внедрение зависимостей:** dishka
- **Валидация данных:** Pydantic

**Сервисы:**
- **PostgreSQL:** Основная база данных для постоянного хранения.
- **Redis:** Используется в качестве брокера сообщений для Taskiq и для кэширования.

## Фронтенд (TypeScript/React)

Фронтенд представляет собой современное одностраничное приложение.

**Основные библиотеки:**
- **Фреймворк:** React 19
- **Инструмент сборки:** Vite
- **Маршрутизация:** TanStack Router
- **Получение данных и состояние:** TanStack Query
- **Формы:** TanStack Form
- **HTTP-клиент:** Axios
- **Компоненты пользовательского интерфейса:** @maxhub/max-ui
- **Стилизация:** Tailwind CSS

**Инфраструктура:**
- **Веб-сервер:** Nginx (для обслуживания собранных статических файлов)

## Инфраструктура и оркестрация

- **Контейнеризация:** Docker
- **Оркестрация:** Docker Compose
